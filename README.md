###Minimal Requirements
* PHP 7.1.3
* NodeJS 6.10.3
* npm
* composer

###Install
1. `git clone git@github.com:biforb4/clickmeeting-paypal.git`
2. `composer install`
3. `npm install`
4. `npm build`

### Run
Run the app with the built-in php server:

`php -S localhost:9000 -t "./public"`

and browse to http://localhost:9000
